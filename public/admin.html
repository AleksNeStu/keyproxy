Telegram Bot

How to set up:

  1. Open Telegram and search for @BotFather
  2. Send /newbot and follow the prompts
  3. Copy the bot token and paste it above
  4. To get your User ID, search for @userinfobot on Telegram and send it any message
  5. Add your User ID to the allowed list (or leave empty to allow anyone)

Bot commands:

/models - Select provider & model

/clear - Clear conversation history

/logs - View recent API logs

/status - Show current model info
